All notable changes to the Simulation KIT will be documented in this file.

## [0.1.0] - 2026-02-04

### Added

- Initial release of the Simulation KIT
- Adoption view with vision, mission, and use cases
- Software development view with API specifications
- Semantic models for SimulationRequest and SimulationResult
- Reference architecture for federated simulation
- Basic implementation examples for provider and consumer
- Integration guide with EDC connectors
- Logo and branding assets

### Documentation

- Created comprehensive adoption view for business stakeholders
- Developed technical software development view for implementers
- Added architecture diagrams and data flow illustrations
- Included API specifications with examples
- Provided Python code samples for quick start

### Standards

- Aligned with Catena-X data exchange principles
- Integrated with Connector KIT for sovereign data sharing
- Compatible with Digital Twin KIT for asset representation
